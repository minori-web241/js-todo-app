2. React との違いを理解する（技術進化を体感することが目的）

## デモ
<img src="./todo-app-js.gif" width="500">

## 使用技術
- HTML
- CSS
- JavaScript (Vanilla JS)

## 実装機能
- TODO の追加
- 削除
- 完了 → 完了リストへ移動
- 戻す → 未完了リストへ戻す
